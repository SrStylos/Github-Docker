## Para poder descargarte este proyecyo te tienes que ir a mi repositorio público https://github.com/SrStylos/Github-Docker donde se encuentra todo lo disponible para su ejecución, para poder iniciar todo debes tener el interprete de git en tu dispositivo para ello te puedes ir a la página web https://git-scm.com/install/windows y lo instalas, utiliza el archibo .bat llamado buiild-github-docker.bat para construit la imagen y run-github-docker.bat para iniciar la página y para pararlo utilizas el archivo stop-github-docker.bat. Para comprobar que todo esta funcionando, en tu navegador ingresas a http://localhost:8080/ y tendrias que ver la informacion de la página web.
